> [!NOTE]
> Em Python, os m√©todos `append()` e `extend()` s√£o usados para adicionar elementos a uma lista, mas funcionam de formas diferentes. Veja a compara√ß√£o clara:
> 
> ---
> 
> ### üîπ `append()`
> 
> - **O que faz**: Adiciona **um √∫nico elemento** ao final da lista.
>     
> - **Mesmo que o elemento seja uma lista**, ele ser√° adicionado como **um √∫nico item**.
>     
> 
> **Exemplo:**
> 
> ```python
> lista = [1, 2, 3]
> lista.append([4, 5])
> print(lista)  # Resultado: [1, 2, 3, [4, 5]]
> ```
> 
> ---
> 
> ### üîπ `extend()`
> 
> - **O que faz**: Adiciona **cada elemento de um iter√°vel (como lista, tupla, etc.)** separadamente √† lista original.
>     
> - **√â como "desembrulhar" e adicionar os itens individualmente.**
>     
> 
> **Exemplo:**
> 
> ```python
> lista = [1, 2, 3]
> lista.extend([4, 5])
> print(lista)  # Resultado: [1, 2, 3, 4, 5]
> ```
>
